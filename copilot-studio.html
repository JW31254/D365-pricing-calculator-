Copilot Studio Pricing Calculator

This calculator helps you estimate the message consumption and cost for a single Copilot Studio agent. Enter the number of each action type in your agent flow and the estimated monthly runs to calculate total message usage.

Note: Interactive use by authenticated Microsoft 365 Copilot users within M365 apps/services is included at no extra cost.

Usage Examples

Customer Support Agent: 4 classic answers, 2 generative answers per run. With 900 customers per day: [(4×1)+(2×2)] × 900 = 7,200 messages per day.

Sales Performance Agent: 4 generative answers, 4 tenant graph groundings per run. With 100 users: [(4×2)+(4×10)] × 100 = 4,800 messages per day.

Order Processing Agent: 4 agent actions per run. Each order uses [(4×5)] = 20 messages.
